This project is a Stockfish-powered chess bot trained to reflect my own playing style rather than feel like playing against an engine.
<br><br>
It uses <strong>Stockfish</strong> as the engine, <strong>chess.js</strong> for game logic, and <strong>chessboard.js</strong> for the board interface, with chess piece graphics based on the Wikipedia chess piece set and UI icons from Font Awesome.
<br><br>
The bot is designed to play around my own rating and is intended to feel more personal than a standard engine opponent. The goal is not just to create a strong bot, but one that feels closer to playing against me.
<br><br>
I’m continuing to expand the project by experimenting with conversational features and a personality inspired by my own approach to chess. As the project develops, I plan to keep refining the experience and exploring new ways to make it feel more natural and distinctive.
<br><br>
If you want to try it, start a game and see how it plays. If you’d like to play a real game against me, click my username at the top of the board.
